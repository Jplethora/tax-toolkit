# Jerisma Plethora Site

This folder contains a static site that is ready for GitHub Pages.

## Publish

1. Upload the contents of this folder to the root of your GitHub Pages repository.
2. Enable GitHub Pages for the repository if it is not already enabled.
3. Confirm the published URL, then test the internal links and calculator pages.
